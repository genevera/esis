Elastic Search Index & Search

* Free software: MIT license
* Documentation: https://esis.readthedocs.org.

Features
--------

* Index content for every SQLite database row in elasticsearch
* Search indexed content

Usage
-----

.. code-block:: bash

    esis index <directory>
    esis search <query>
